# 🚀 Setup React + TypeScript App

---

## 1. Install Node.js

* Download from [https://nodejs.org](https://nodejs.org)
* Verify:

  ```bash
  node -v
  npm -v
  ```

---

## 2. Create Project with Vite (Recommended)

```bash
npm create vite@latest my-react-app
```

* Choose framework: **React**
* Choose variant: **TypeScript + SWC** (faster compiler)

Then:

```bash
cd my-react-app
npm install
```

---

## 3. Project Structure

```
my-react-app/
│── index.html
│── tsconfig.json
│── vite.config.ts
│── package.json
│── src/
│   ├── main.tsx
│   ├── App.tsx
│   ├── components/
│   │    └── Hello.tsx
│   ├── styles/
│   │    └── index.css
```

---

## 4. Start Dev Server

```bash
npm run dev
```

Visit → `http://localhost:5173`

---

## 5. Example Code

### `src/App.tsx`

```tsx
import { useState } from "react";
import Hello from "./components/Hello";

function App() {
  const [count, setCount] = useState<number>(0);

  return (
    <div style={{ textAlign: "center", padding: "2rem" }}>
      <h1>🚀 React + TypeScript App</h1>
      <Hello name="Hasanthika" />
      <p>Counter: {count}</p>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}

export default App;
```

### `src/components/Hello.tsx`

```tsx
type HelloProps = {
  name: string;
};

export default function Hello({ name }: HelloProps) {
  return <h2>Hello, {name} 👋</h2>;
}
```

---

## 6. Add TypeScript Types

TypeScript is already configured by Vite. You can adjust `tsconfig.json` if needed:

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "lib": ["DOM", "ES2020"],
    "jsx": "react-jsx",
    "strict": true,
    "moduleResolution": "node",
    "esModuleInterop": true
  }
}
```

---

## 7. Useful Dev Tools

* **React Router** (routing):

  ```bash
  npm install react-router-dom
  npm install --save-dev @types/react-router-dom
  ```
* **State management** (optional):

    * Redux Toolkit: `npm install @reduxjs/toolkit react-redux`
    * Or Zustand, Jotai, Recoil depending on preference.
* **UI frameworks**:

    * Tailwind CSS (`npm install -D tailwindcss postcss autoprefixer && npx tailwindcss init -p`)
    * Material UI: `npm install @mui/material @emotion/react @emotion/styled`

---

## 8. Build for Production

```bash
npm run build
npm run preview
```

This generates an optimized build in the `dist/` folder.
